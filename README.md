# E-Commerce React

## ¿De qué se trata?

Un proyecto de tienda en línea (e-commerce) desarrollado con React. Los usuarios pueden explorar productos, agregarlos al carrito y realizar compras. El carrito se gestiona con **Context API** y los datos se almacenan en **Firebase**.

## Tecnologías utilizadas

- **React 19**: Framework de UI
- **Vite**: Bundler y servidor de desarrollo
- **Firebase**: Base de datos Firestore
- **React Router**: Navegación entre páginas
- **React Icons**: Iconos SVG
- **CSS3**: Estilos

## Instalación

1. Clonar el proyecto
   ```bash
   git clone https://github.com/cascallanaleo/ecommerce-LEC.git
   cd ecommerce-LEC
   ```

2. Instalar dependencias
   ```bash
   npm install
   ```

3. Iniciar el servidor
   ```bash
   npm run dev
   ```

4. Abrir en el navegador
   ```
   http://localhost:5173
   ```

## Características principales

- Catálogo de productos
- Detalle de cada producto
- Carrito de compras con totales
- Eliminar productos del carrito
- Proceso de checkout
- Órdenes guardadas en Firebase
